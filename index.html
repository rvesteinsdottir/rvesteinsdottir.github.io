About Me:

Hi! My name is Raisah (she/her), I'm a student at Ada Developers Academy and I'm a nonprofit operations manager turned software engineer. I applied for Ada because programming allows me to combine my love of problem solving with my deep commitment to the Seattle community. Explore this website, you'll learn a little bit about my coding projects, background and interests.

I just developed my own iOS color gradient game using voronoi diagram partitioning, check out my project at raisah.itch.io/spectrum.

I've been a software development intern at Milliman since February. I work in C#, JavaScript, and React.js and my recent projects have included developing an Azure DevOps Continuous Integration YAML pipeline, building out new APIs, and churning out bug fixes.
