portfolio

Projects:

Spectrum

Soothing color gradient game built using Unity Game Engine. Each puzzle board is unique: polygons are created using voronoi diagram partitioning (center points are randomly sampled, all other points of the plane corresponds to the region of the closest center point). Colors can be selected by user for extra levels of difficulty.

Sudsy

Built an e-commerce site from scratch with a team of student developers. Sudsy is hosted on Heroku and features user authentication, merchant-only pages and dashboards and an easy to use checkout.

Hotel

Designed hotel booking system using object-oriented design principles, created classes and class methods, utilized test driven development.

Rideshare

Implemented rideshare system, collaborated through pair programming, worked with files and large amounts of data.
